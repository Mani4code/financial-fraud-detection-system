# Financial Fraud Detection System

This project contains a financial fraud detection pipeline with a Streamlit dashboard, model training assets, and supporting database utilities.

## Project Structure

- app/ - Application code and dashboard entry points
- database/ - Database schemas, migrations, and storage helpers
- model/ - Training scripts, model artifacts, and feature engineering logic
- requirements.txt - Python dependencies
- Dockerfile - Container image definition
- docker-compose.yml - Local container orchestration
